## 运行环境
- PHP 7.0+ (v2.8.0 开始 >= 7.1.3)
- composer

> php5 请使用 v1.x 版本[https://github.com/yansongda/pay/tree/v1.x](https://github.com/yansongda/pay/tree/v1.x)

<a name="安装方式"></a>

## 安装方式
```shell
composer require yansongda/pay -vvv
```
